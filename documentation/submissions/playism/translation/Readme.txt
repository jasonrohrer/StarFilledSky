『星空の中へ』

無限に繰り返される戦術シューティング（製作：Jason Rohrer）

http://insideastarfilledsky.net

このゲームではマウスによる照準が重要な役割を果たします。
可能であればマウスを接続してください。

状況に応じてBGMが微妙に変化します。
スピーカーかヘッドフォンでお楽しみください。


== 操作方法 ==

[P]または[Esc]キーを押すとポーズがかかり、
メニュースクリーンが表示されます。

[F]キーを押すとフラッグを編集できます。

ゲームは簡単なキーボードとマウス操作でプレイでき、
ゲーム中に簡単な操作説明が表示されます。
操作説明は[T]キーでいつでも再表示できます。

お使いのキーボードの[W、A、S、D]キーで上手く移動できない場合は、
『settings/upDownLeftRightKeys.ini』をテキストエディタで編集して
4つのキーを自由に設定にすることができます。
矢印キーと[I、J、K、L]は最初から移動キーのオプションとして
割り当てられているので注意してください。


== 画面サイズ ==

お使いのモニターでデフォルトの640x480フルスクリーンがうまく表示されない場合は、
『settings』フォルダで画面サイズを編集することができます。
また、フルスクリーンモードに切り替えることもできます。
フルスクリーンモードでゲームを開始した後は、
[Alt + Enter]キーを押すとウィンドウモードに切り替わります。

デフォルトの60FPSでゲームの動作が遅くなる場合
（特にCPUに負担がかかる、何かに侵入する操作の時など）は、『settings/halfFrameRate.ini』に「1」を入力して
30FPSに切り替えてみてください。

30FPSでも動作が遅い場合は、同じ設定ファイルに『2』を入力して
15FPSに切り替えてみてください。
それでも動作が遅い場合は、同じ設定ファイルに『3』を入力して
7.5FPSに切り替えてみてください。
